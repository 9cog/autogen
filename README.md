| Concept | Meaning | Etymology |

| **Autostasis** | self-stabilising | *auto* + *stasis* |
| **Morphogenesis** | self-shaping | *morphē* "form" + *genesis* "becoming" |
| **Homeostasis** | self-sustaining | + *homoios* + *stasis* |
| **Autopoiesis** | self-producing | + *poiesis* "making" |
| **Autognosis** | self-knowing | + *gnōsis* "knowing" |
| **Autogenesis** | self-creating | + *genesis* "origin" |
| **Autonomos** | self-governing | + *nomos* "law" |
| **Teleogenesis** | self-directing | *telos* "purpose" + *genesis* |
